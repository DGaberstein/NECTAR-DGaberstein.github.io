# NACTAR

Static website for NACTAR, built with plain HTML, CSS, and JavaScript.

## Pages

- Home: index.html
- Products: produkte.html
- Cultivation: anbau.html
- Contact: kontakt.html
- Terms (AGB): agb.html
- Imprint: impressum.html
- Privacy: datenschutz.html

## Structure

- HTML: page files in the root directory
- Styles: style.css
- Scripts: script.js
- Images: images/

## Run locally

Open index.html in a browser, or use a simple local server if you prefer.

## Edit

- Update layout and content in the HTML files
- Adjust styles in style.css
- Add interactivity in script.js
